Class: EventEmitter

EventEmitter()

EventEmitter class to manage event-driven architecture.

Constructor

new EventEmitter()

Creates an instance of EventEmitter. This constructor initializes the callbacks object, which is used to store event listeners. The `base` namespace is initialized as an empty object to hold default event listeners.
